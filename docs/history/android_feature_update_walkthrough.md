# Android 기능 업데이트 (iOS 스타일 적용)

## 서문
사용자 요청에 따라 안드로이드 앱의 UI와 기능을 대폭 개선하여 iOS 버전과 경험을 통일했습니다.

## 🌟 주요 기능 업데이트

### 1. 물풍선(Callout) 디자인 전면 개편
- **기존:** 단순 텍스트 목록의 투명 캔버스 드로잉.
- **변경:**
    - **디자인:** `[아이콘] [시간/제목] [휴지통]` 순서의 깔끔한 행(Row) 기반 레이아웃.
    - **스타일:** 흰색 배경에 그림자와 회색 테두리 적용, 모서리를 둥글게 처리 (Rounded).
    - **유연성:** 표시 개수가 많으면 스크롤 가능하며, 최대 표시 개수를 설정에서 조절 가능.

### 2. 내 정보 (User Profile) 개선
- **기존:** 단순 기능 없음.
- **변경:**
    - **위치:** 오른쪽 아이콘들을 가리지 않는 좌측 오버레이 형태로 변경.
    - **정보:** 닉네임, 전화번호 표시 (Design Only).
    - **설정 기능:**
        - **물풍선 최대 개수:** +/- 버튼으로 조절 가능.
        - **글씨 크기:** 작게/중간/크게 선택 가능.

### 3. 타임 트래블 (어제로 가기)
- **기능:** 알림 아이콘 왼쪽에 **[시계(되감기)]** 버튼 추가.
- **동작:**
    - **기본 모드:** 현재 기준 ±24시간 내의 핀만 표시.
    - **어제 모드 (활성화 시):** 버튼이 빨간색으로 변하며, **어제(24~48시간 전)**의 핀들만 필터링하여 지도에 표시.
    - **UI 제어:** 어제 모드 진입 시 알림, 로그인 등 현재 시점 관련 버튼 비활성화(Dimmed).

## 🛠️ 기술적 변경 사항
- **`UserProfileView.kt`:** 설정 UI를 포함한 새로운 컴포넌트 생성.
- **`TodoViewModel.kt`:** UI 설정을 위한 상태 변수(`maxPopupItems`, `popupFontSize`) 추가.
- **`MainScreen.kt`:** 
    - 시간 필터링(`minTime` ~ `maxTime`) 로직 구현.
    - `Canvas` 드로잉 코드를 제거하고 Compose `LazyColumn`으로 교체.

> [!TIP]
> 이제 안드로이드에서도 아이폰과 동일한 수준의 세련된 지도 사용 경험과 "어제의 기록"을 확인하는 강력한 기능을 즐길 수 있습니다. 🚀
